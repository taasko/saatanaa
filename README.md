# Säätänää

## Features

- Real-time infernal weather updates
- Stylish meme-inspired design

## How to Use

1. Enter your city or try geolocation
2. Enjoy Säätänään accurate weather forecasts!

## Development

### Requirements

- Docker (Compose)

Start development server:

```shell
docker compose up -d
```

Navigate to: https://saatanaa.localhost/
